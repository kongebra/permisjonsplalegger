# Jonas (29)

## Profil
- **Rolle:** Far
- **Familiesituasjon:** Samboer med Ida (28), venter første barn. Jonas er utvikler, Ida er sykepleier.
- **Funksjonsnedsettelse:** Ingen
- **Teknisk kompetanse:** Svært høy
- **Personlighet:** Nysgjerrig, detaljorientert, litt nerdet

## Kontekst
Jonas vil forstå *alt* om hvordan kalkulatoren fungerer. Han åpner DevTools for å se nettverksforespørsler (og blir fornøyd når det ikke er noen), inspiserer koden, og vil vite hvilke formler som brukes. Stiller spørsmål ingen andre tenker på.

## Typisk bruksmønster
- Inspiserer nettverkstrafikk og localStorage
- Tester edge cases (hva skjer med 0 kr i lønn? 10M kr?)
- Leser hjelpetekster for å verifisere at de er korrekte
- Verdsetter transparent metodikk og kildehenvisninger
- Kan gi tilbakemelding om bugs eller forbedringsforslag
- Liker å se "vis beregning" eller liknende detaljvisninger

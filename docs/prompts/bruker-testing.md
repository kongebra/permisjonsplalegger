Kjør brukertesting med 5 tilfeldige personas fra docs/personas/.

**Oppsett:**

- App kjører på localhost:3001
- Bruk Playwright MCP for interaksjon

**Testflyt:**

1. Start fra onboarding (forsiden)
2. Fullfør wizard-stegene
3. Naviger til kalenderen via naturlig flyt
4. Utforsk kalender-funksjonen

**Fokusområder:**

- Onboarding-opplevelse (forståelighet, flyt)
- Kalender-funksjonalitet (drag-seleksjon, periodevising)
- Eventuelle problemer eller forvirring

**Output:**
Samlet rapport med funn fra alle personas, inkludert:

- Hva fungerte bra
- Smertepunkter/forvirring
- Forbedringsforslag
